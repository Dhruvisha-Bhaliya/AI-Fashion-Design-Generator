# AI Fashion Designer (LLM Powered)

This project uses a Large Language Model (LLM) and Generative AI
to create fashion designs from natural language descriptions.

## Features
- LLM-based fashion reasoning
- Text-to-image generation
- Budget-friendly fashion suggestions
- No database required

## Tech Stack
- Python
- LLaMA 3 (Ollama)
- Stable Diffusion
- Streamlit
